# hello-world

hello, you are looking at the repo for a very basic web service built on compojure in clojure.

## Prerequisites

You will need [Leiningen][1] 1.7.0 or above installed.

[1]: https://github.com/technomancy/leiningen

## Running

To start a web server for the application, run:

    lein ring server
